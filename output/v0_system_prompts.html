v0 系统提示词

Vercel 的 AI 助手系统

探索 v0 的强大功能与设计理念

v0 是 Vercel 的 AI 驱动助手，专为开发者设计，旨在提供最先进的代码生成和技术支持。它采用 MDX 格式响应，允许嵌入 React 组件，并默认使用 Next.js App Router 架构。

核心功能

代码项目生成

使用 <CodeProject> 标签组织文件，渲染 React 和全栈 Next.js 应用。自动推断 npm 模块，预装 Tailwind CSS、Next.js 和 shadcn/ui 组件。

图像与媒体处理

使用占位图像、Lucide React 图标包，支持 3D 模型和音频文件。在 canvas 上渲染图像时自动设置 crossOrigin 为 "anonymous"。

图表与数学公式

使用 Mermaid 语言渲染图表和流程图，用 LaTeX（包裹在双美元符号中）表示数学公式，增强内容的可视化表达。

快速编辑

使用 <QuickEdit /> 组件对现有代码块进行小修改，适合进行少量代码行的更改，提高编辑效率。

Node.js 执行环境

使用 Node.js 执行块运行脚本，演示算法或处理数据。支持 ES6+ 语法和内置 fetch，自动安装所需的第三方库。

无障碍设计

实现无障碍最佳实践，使用语义化 HTML 元素和正确的 ARIA 角色/属性，为屏幕阅读器提供专用文本支持。

代码示例

React 组件示例

function ChatRoom({ roomId }) {
  useEffect(() => {
    const connection = createConnection(roomId);
    connection.connect();
    return () => connection.disconnect();
  }, [roomId]);
  // ...
}

使用 React 的 useEffect 钩子连接到外部系统，实现组件与外部资源的同步。

自定义钩子示例

function useChatRoom({ serverUrl, roomId }) {
  useEffect(() => {
    const options = {
      serverUrl: serverUrl,
      roomId: roomId
    };
    const connection = createConnection(options);
    connection.connect();
    return () => connection.disconnect();
  }, [roomId, serverUrl]);
}

将 Effect 逻辑封装在自定义钩子中，提供更具声明性的 API，简化组件代码。

AI 集成

v0 使用来自 sdk.vercel.ai 的 AI SDK 和工具，通过 'ai' 和 '@ai-sdk' 包提供 AI 功能。以下是使用 AI SDK 生成文本的示例： 

import { generateText } from "ai"
import { openai } from "@ai-sdk/openai"
const { text } = await generateText({
  model: openai("gpt-4o"),
  prompt: "What is love?"
})

AI SDK 标准化了跨支持提供商集成人工智能模型的过程，使开发者能够专注于构建出色的 AI 应用，而不是浪费时间在技术细节上。
